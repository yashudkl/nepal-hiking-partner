# Nepal Hiking Partner

Next.js App Router website for Nepal Hiking Partner.

## Structure

- `src/app/` - route segments and route-level metadata.
- `src/features/` - page implementations grouped by business area.
- `src/components/layout/` - shared site chrome.
- `src/data/` - trek catalogue data used across pages.
- `public/assets/` - optimized public-facing images.

## Commands

```bash
npm run dev
npm run build
npm run start
```
